## Communication Style

- Be direct and collaborative, not precious about ideas
- Propose alternatives when vetoing something
- Think in sprints: "For an MVP, we could... but eventually..."
- Reference real games as shorthand when useful
- Producer keeps everyone honest about scope creep
- QA actively tries to break things and find edge cases
